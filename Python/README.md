## 📊 Data Analysis Skills with pandas

### 📥 Loading & Inspecting Data
- Imported CSV datasets into pandas DataFrames  
- Explored structure using `.head()`, `.info()`, `.describe()`  

### 🔍 Filtering & Subsetting
- Used `.loc[]` for label‑based selection  
- Used `.iloc[]` for position‑based selection  
- Applied conditional filtering to isolate:
  - High‑value sales  
  - Specific product categories  
  - Missing or inconsistent entries  

### 🧹 Data Cleaning & Preparation
- Removed missing values with `.dropna()`  
- Filled missing values using `.fillna()`  
- Renamed columns and changed data types  
- Sorted data using `.sort_values()`  

### 🔄 Data Manipulation
- Grouped data using `.groupby()` to calculate:
  - Total sales  
  - Average values  
  - Category‑level summaries  
- Created new calculated columns  
- Merged datasets where needed  

### 📈 Data Visualisation
Used pandas built‑in plotting and libraries such as **matplotlib** to create:

- Bar charts  
- Line charts  
- Histograms  
- Scatter plots  

These visualisations helped uncover trends in retail and sales data, such as:

- Best‑selling products  
- Seasonal patterns  
- Customer behaviour  
- Category‑level performance
